This is the string script that goes into the jmolApplet() function. The plug-in uses this script to compose the Javascript function jmolApplet($WIDTH, "load FILES $FILE;$SCRIPT);");.

Add one statement per line. So instead of "model *; select all; wireframe off;", you would have:
model *
select*
wireframe off
Notice that you don't need to add the ; in the end of the statement too.
